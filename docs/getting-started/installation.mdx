## Add to Your Project

Add Ack to your project using the Dart CLI:

```bash
# For Dart projects
dart pub add ack

# For Flutter projects
flutter pub add ack
```

Or manually add to your `pubspec.yaml` (check [pub.dev](https://pub.dev/packages/ack) for the latest version):

```yaml
dependencies:
  ack: ^1.0.0 # Replace with latest version
```

## Code Generator (Experimental)

The `ack_generator` package is experimental and not recommended for production use. For production applications, use manual schema definition with the current Ack API.

If you want to experiment with the code generator:

```bash
# Add the experimental generator and build_runner
dart pub add --dev ack_generator build_runner
```

**Note:** The generated code may not compile with the current ack package version. Manual schema definition is the recommended approach.

## Basic Usage after Installation

Import Ack in your Dart files:

```dart
import 'package:ack/ack.dart';

// 1. Define a simple schema (e.g., for a name)
final nameSchema = Ack.string().minLength(3);

// 2. Validate some data against the schema
final result = nameSchema.validate('John');

// 3. Check the result
if (result.isOk) {
  print('Valid: ${result.getOrThrow()}'); // Access the valid data
} else {
  print('Invalid: ${result.getError()}'); // Get the error details
}
```

See the [Quickstart Tutorial](./quickstart-tutorial.mdx) for a more complete example.
