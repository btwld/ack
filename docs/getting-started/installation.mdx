## Add to Your Project

Add Ack to your project using the Dart CLI:

```bash
# For Dart projects
dart pub add ack

# For Flutter projects
flutter pub add ack
```

If you edit `pubspec.yaml` manually, copy the current `ack` version from
[pub.dev](https://pub.dev/packages/ack) or use the dependency line generated by
the CLI.

## Code Generator (`@AckType()`)

If you want typed wrappers for hand-written schemas, add the annotation and
generator packages alongside `ack`:

```bash
dart pub add ack ack_annotations
dart pub add --dev ack_generator build_runner
```

If you edit `pubspec.yaml` manually, copy current versions from
[ack](https://pub.dev/packages/ack),
[ack_annotations](https://pub.dev/packages/ack_annotations), and
[ack_generator](https://pub.dev/packages/ack_generator), then keep
`ack_generator` and `build_runner` under `dev_dependencies`.

`ack_generator` does not generate schemas from classes. It reads top-level Ack
schema variables and getters annotated with `@AckType()` and emits typed
extension wrappers with `parse()` / `safeParse()` helpers.

```dart
import 'package:ack/ack.dart';
import 'package:ack_annotations/ack_annotations.dart';

part 'user.g.dart';

@AckType()
final userSchema = Ack.object({
  'name': Ack.string(),
  'email': Ack.string().email(),
});
```

Run the generator with:

```bash
dart run build_runner build --delete-conflicting-outputs
```

See the [generator documentation](https://docs.page/btwld/ack) for more
AckType examples and supported schema shapes.

## Basic Usage after Installation

Import Ack in your Dart files:

```dart
import 'package:ack/ack.dart';

// 1. Define a simple schema (e.g., for a name)
final nameSchema = Ack.string().minLength(3);

// 2. Validate some data against the schema
final result = nameSchema.safeParse('John');

// 3. Check the result
if (result.isOk) {
  print('Valid: ${result.getOrThrow()}'); // Access the valid data
} else {
  print('Invalid: ${result.getError()}'); // Get the error details
}
```

See the [Quickstart Tutorial](./quickstart-tutorial.mdx) for a more complete example.
