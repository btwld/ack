## Add to Your Project

Add Ack to your project using the Dart CLI:

```bash
# For Dart projects
dart pub add ack

# For Flutter projects
flutter pub add ack
```

Or manually add to your `pubspec.yaml` (check [pub.dev](https://pub.dev/packages/ack) for the latest version):

```yaml
dependencies:
  ack: ^1.0.0 # Replace with latest version
```

## Code Generator (Production Ready)

The `ack_generator` package provides automatic schema generation from Dart classes, reducing boilerplate code and improving type safety.

Add the generator and build_runner to your project:

```bash
# Add the generator and build_runner
dart pub add --dev ack_generator build_runner
```

Or add to your `pubspec.yaml`:

```yaml
dev_dependencies:
  ack_generator: ^1.0.0 # Replace with latest version
  build_runner: ^2.4.0
```

**Features:**
- Automatic schema generation from annotated classes
- Support for complex types including discriminated unions
- Comprehensive error handling with detailed messages

See the [generator documentation](https://docs.page/btwld/ack) for usage examples.

## Basic Usage after Installation

Import Ack in your Dart files:

```dart
import 'package:ack/ack.dart';

// 1. Define a simple schema (e.g., for a name)
final nameSchema = Ack.string().minLength(3);

// 2. Validate some data against the schema
final result = nameSchema.safeParse('John');

// 3. Check the result
if (result.isOk) {
  print('Valid: ${result.getOrThrow()}'); // Access the valid data
} else {
  print('Invalid: ${result.getError()}'); // Get the error details
}
```

See the [Quickstart Tutorial](./quickstart-tutorial.mdx) for a more complete example.
